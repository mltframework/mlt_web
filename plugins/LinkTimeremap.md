## Parameters

### map

title: Map    
description:
A map of input frame times to output frame times.  
type: float  
readonly: no  
required: no  
animation: yes  

### image_mode

title: Image Mode    
description:
The image mode to use.  
type: string  
readonly: no  
required: no  
values:  

* nearest
* blend

### speed

title: Speed    
description:
The instantaneous speed of the last frame that was processed.  
type: float  
readonly: yes  
required: no
